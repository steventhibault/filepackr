FAQ

Scan to folder questions.

Does this require changing scanners?

No. filepackR is focused on the workflow after scanning, using scanned files and folders your team already works with.

Can this help with shared folders?

Yes. The workflow can be shaped around existing local or shared Windows folder structures.

Is this the same as basic PDF splitting?

No. Splitting is only one step. filepackR also supports recognition, review, naming, routing, and exception handling.
